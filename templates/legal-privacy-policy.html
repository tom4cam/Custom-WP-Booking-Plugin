Third-Party Service Providers

To run the booking system we rely on a small number of trusted processors. Each receives only what's needed to perform its function:

  • Square — processes card payments. Card details go directly to Square's servers; we never touch them. See Square's Privacy Notice.
  • Twilio — delivers appointment confirmation and reminder text messages. See Twilio's Privacy Notice.
  • Google Calendar — stores the appointment so {practitioner_name} can see the schedule. The event title and time go to Google; payment information does not. See Google's Privacy Policy.
  • Venmo — only if you choose Venmo as your payment method. Venmo handles the transaction; we see only that payment was confirmed. See Venmo's Privacy Policy.
  • Web hosting — the WordPress hosting provider stores the site database, which contains the records described above. Database access is restricted to {practitioner_name} and authorized administrators.
